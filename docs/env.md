# Environment Variables

## App (Expo)
- `EXPO_PUBLIC_SUPABASE_URL` – Supabase project URL.
- `EXPO_PUBLIC_SUPABASE_PUBLISHABLE_KEY` – Supabase publishable key for client-side access.
- `EXPO_PUBLIC_EXPO_PROJECT_ID` – Expo project ID for push tokens.

## Seed Script / Admin Tools
- `SUPABASE_URL` – Supabase project URL.
- `SUPABASE_SECRET_KEY` – Supabase secret key for admin operations.

## Optional
- `SUPABASE_JWT_SECRET` – Only needed for custom auth tooling.
