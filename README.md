# Beer Snap MVP

Minimalist, camera-first beer photo sharing app for iOS, Android, and Web.

## Repo Structure
- `apps/beer-snap` – Expo app (iOS/Android/Web)
- `packages/domain` – Domain types
- `packages/data` – Supabase client + queries
- `packages/ui` – Design system
- `supabase` – Migrations, seed script, edge functions
- `docs` – Architecture and operational docs
